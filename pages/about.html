关于我们

了解 OpenLight Studio 的故事

📅

2022.12.01

OpenLight Studio 正式成立。这是两个充满激情的团队——Open 工作室与闪电工作室的历史性合并。我们相信，协作能创造更大的价值。

🎯

使命

致力于底层技术的探索与教育普及。从操作系统内核到青少年编程语言，我们试图构建更明亮、更易懂的计算世界。

💡

愿景

让每一个对计算机科学感兴趣的人都能轻松入门，让复杂的技术变得触手可及。我们用代码照亮前行的道路。

🌍

开源精神

我们坚信开源的力量。所有项目均以开源形式发布，欢迎全球开发者参与贡献，共同推动技术进步。

发展历程

2022.12

OpenLight Studio 成立

组织在 GitHub 上创建，开始探索底层技术

BrightS

BrightS 内核项目启动

基于 UNIX V6 设计理念的 x86_64 UEFI 内核，使用 C 语言开发，已完成 21 次提交。支持 UEFI 启动、串口控制台、NVMe/AHCI 存储探测、Btrfs 文件系统挂载

D--

D Substract (D--) 编程语言

专为青少年设计的编程语言，使用 C++ 开发，已完成 10 次提交。MIT 开源许可证

FH Clac

Frequency Histogram Clac 发布

跨平台直方图生成器，使用 C++ 开发，已完成 16 次提交，获得 2 Stars。v1.0.2 版本于 2026 年 3 月发布，支持 Linux、macOS、Windows
